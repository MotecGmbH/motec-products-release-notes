Release Notes MCDE3090a-HDMI Software Version 1.0.0
=========================================

To report a bug please send an email to motec.bugreport@ametek.com

new features
------------

- streaming
  - two rtp streams
  - RTSP streaming possibility
  - two http streams
  - hdmi stream

- website
  - live video of http streams
  - camera: mirror horizontal, mirror vertical, stream test image, avoid flickr, exposure, saturation, aperture, mcp port
  - streams: drop down stream selection, activate stream, http stream path, activate rtp, stream destination ip and port, fps, compression, cropping
  - network: auto ip assignment, ip address, prefix length, gateway, dns, mac address
  - service: firmware update, reboot, upload/download configuration, system status, factory reset
  
- mcp: please ask for detailed command documentation
